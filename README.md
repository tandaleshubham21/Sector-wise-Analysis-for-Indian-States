# Sector-wise-Analysis-for-Indian-States

Project Overview
This project focuses on analyzing district-wise sectoral economic performance across Indian states using SQL. The dataset includes economic contributions from the Primary, Secondary, and Tertiary sectors at both constant and current prices.

The objective is to identify trends, compare district performance, and generate meaningful insights using SQL queries and analytical techniques.

Objectives
- Analyze sector-wise economic growth across districts and states
- Compare district and state economic performance
- Perform trend and time-series analysis
- Generate business insights using SQL
- Practice advanced SQL concepts on real-world datasets

Tools & Technologies Used
- SQL
- MySQL / PostgreSQL
- DBeaver / SQL Workbench
- Power BI / Tableau
- GitHub

Dataset Information
The dataset contains:
- State Name
- District Name
- Year
- Primary Sector Contribution
- Secondary Sector Contribution
- Tertiary Sector Contribution
- Total Current Prices
- Total Constant Prices
- Per Capita Current Prices

SQL Concepts Used
- SELECT Statements
- WHERE Filtering
- GROUP BY
- ORDER BY
- Aggregate Functions
- Window Functions
- Subqueries
- Ranking Functions
- Joins
- Time-Series Analysis

Analysis Performed

1. State-wise Sector Analysis
Analyzed economic contributions across different Indian states.

2. Growth Analysis
Calculated year-on-year growth in sector contributions.

3. District Ranking
Ranked districts based on total economic contribution.

4. Trend Analysis
Identified districts with the highest tertiary sector growth.

5. Comparative Analysis
Compared per capita contributions between districts

Key Insights
- Tertiary sector contribution is significantly higher in urban districts.
- Some districts remain heavily dependent on the primary sector.
- Economic growth varies across states and districts.
- Per capita current prices are higher in economically developed regions.
